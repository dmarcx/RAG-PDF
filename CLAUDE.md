# CLAUDE.md – פרויקט RAG-PDF

## מה הפרויקט עושה
אפליקציית RAG שמאפשרת להעלות קבצי PDF ולשאול שאלות עליהם בעברית ואנגלית.

## סטאק טכנולוגי
| רכיב | טכנולוגיה |
|------|-----------|
| שפה | Python |
| מודל שפה | Anthropic API (`claude-3-5-haiku-20241022`) |
| מסד נתונים וקטורי | ChromaDB |
| קריאת PDF | PyMuPDF |

## כללים לעבודה עם הקוד

1. **הסבר לפני כל שינוי** – לפני כל שינוי בקוד, ציין בדיוק מה אתה עושה ולמה.
2. **הערות בעברית** – כל הערות הקוד (`# ...`) יהיו בעברית.
3. **אל תשנה קבצים קיימים בלי לשאול** – כל שינוי בקובץ קיים מחייב אישור מפורש.
4. **קוד פשוט וקריא** – עדיפות לפשטות ולקריאות על פני חכמה מיותרת.
